# WCMM

WAP Client Management Messenger

## Description

사용자가 자신의 웹사이트에 고객 관리용 메신저를 설치하고 웹 대시보드를 통해서 고객과의 대화를 관리할 수 있는 서비스입니다.

[채널톡 참고](https://channel.io/ko)

[상세 기록](./RECORD.md)

- packages
